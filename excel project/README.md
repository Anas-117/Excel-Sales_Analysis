# ☕ Coffee Shop Sales Dashboard (Excel)

## Project Overview

This project is an **Excel-based interactive dashboard** built to analyze and visualize **Coffee Shop Sales** data. The dashboard provides quick business insights using Pivot Tables, measures, slicers, and KPI cards, designed to fit **100% zoom** across Windows and Mac environments.

---

## Objectives

- Track overall sales performance
- Analyze trends over time
- Identify top-performing products and categories
- Enable quick filtering using slicers
- Present insights in a clean, one-screen dashboard format

---

## Key Features

- **Interactive Dashboard** (Excel)
- **Pivot Tables & Measures** (Data Model)
- **KPI Cards** using shapes linked to measures
- **Line Charts** for trend analysis
- **Slicers** for dynamic filtering
- Designed for **100% Zoom (A1**\*\*:V35\*\*\*\*)\*\* for cross-platform consistency

---

## Tools & Technologies

- Microsoft Excel
- Power Query (data cleaning & transformation)
- Pivot Tables & Data Model
- GETPIVOTDATA formulas
- Excel Shapes for KPI cards

## Insights Generated

### Overall Sales Performance

- **Total Sales:** \$698,812
- Sales show **stable and consistent performance**, indicating steady customer demand.

### Category-Level Insights

- **Top Category:** Coffee
- **Coffee Revenue:** \$269,952 (highest among all categories)
- Coffee is the **primary revenue driver** for the business.

### Store Performance

- **Best Performing Store:** Hell’s Kitchen
- This store contributes the **highest share of total sales**, suggesting higher footfall or stronger customer preference.

### Time-Based Trends

- **Peak Sales Hour:** 10 AM
- Strong morning demand, likely driven by office-goers and breakfast customers.

### Product Performance

- **Top-Selling Product:** Ethiopia (Gourmet Brewed Coffee)
- Premium coffee products perform well, indicating **willingness to pay for quality**.

### Size Preference

- **Most Revenue-Generating Size:** Large
- Followed by Regular
- Customers show a clear preference for **larger-sized beverages**.

### Day-wise Sales Pattern

- **Highest Sales Day:** Monday
- Followed by Friday and Thursday
- Weekdays show consistently strong sales performance.

## Dashboard Design Best Practices Used

- Designed at **100% Zoom**
- Optimized for **Windows Excel constraints**
- No scrolling (single-screen dashboard)
- Clean layout (gridlines, headings hidden)
- Shapes used instead of cells for KPIs
